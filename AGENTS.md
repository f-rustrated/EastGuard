# EastGuard Agent Entry Point

This repo keeps the canonical project instructions in [.agents/AGENTS.md](.agents/AGENTS.md).
Read and follow that file before making code, test, architecture, or documentation changes.

Project-specific skills live in [.agents/skills/](.agents/skills/), which is a symlink to the
canonical skill definitions. When a task matches a skill, read its `SKILL.md` completely before
acting.

Project architectural rules live in [.agents/rules/](.agents/rules/), which is a symlink to the
canonical rule definitions. Before touching a subsystem, read the relevant rule file listed in
`.agents/AGENTS.md`.
